# Meslo Nerd Font Setup

Steadfast is tuned for Meslo Nerd Font, specifically `MesloLGS Nerd Font Mono`.

The font files are available locally at:

```text
/Users/jrxna/github/meslo-nerd-font
```

Install at least:

- `MesloLGSNerdFontMono-Regular.ttf`
- `MesloLGSNerdFontMono-Bold.ttf`

After installing, use this family name where supported:

```text
MesloLGS Nerd Font Mono
```

VSCode settings are included in `../vscode/settings.json`.
