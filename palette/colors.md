# Steadfast Palette

Steadfast is a warm industrial palette for long engineering sessions. It keeps the Gruvbox foundation, lowers visual noise, and leans into tungsten, brass, leather, ash, rust, and old terminal phosphor.

## Core

| Role | Hex | Use |
| --- | --- | --- |
| Background | `#312D2A` | Main editor, writing, and terminal background |
| Deep background | `#282421` | Panels, title bars, code block depth |
| Surface | `#3A3430` | Hover states and subtle elevated areas |
| Surface high | `#45403A` | Focused controls and inactive selections |
| Line | `#3F3832` | Minimal borders and separators |
| Text | `#E5D6B8` | Primary reading and code text |
| Muted text | `#BDAE93` | Secondary text and inactive UI |
| Dim text | `#928374` | Comments, hints, disabled states |
| Accent | `#FACD62` | Cursor, active tabs, key syntax, warm highlights |
| Soft accent | `#D8A657` | Modified state, parameters, subdued brass |

| Role | Hex | Character |
| --- | --- | --- |
| Red | `#CC5C4A` | Errors, deletion, strong keywords |
| Bright red | `#EA6962` | High-signal keyword emphasis |
| Orange | `#D38650` | Control flow and warnings |
| Yellow | `#FACD62` | Functions and primary accent |
| Bright yellow | `#FABD2F` | Terminal bright yellow |
| Green | `#A9B665` | Strings and additions |
| Bright green | `#B8BB26` | Terminal bright green |

| Blue | `#7DAEA3` | Classes, types, links |
| Purple | `#D3869B` | Constants, decorators, regex |

| Leather | `#8F5F3D` | Buttons and grounded warm surfaces |
| Ash | `#7C6F64` | Rulers, graphite details |

## Rules

- Preserve `#312D2A` as the main background.
- Use `#FACD62` sparingly for the cursor and active state clarity.
- Prefer warm contrast over saturation.
- Avoid pure black, glass effects, neon cyan, and high-chroma purple-blue gradients.
- Keep borders low contrast and functional.
